### 欄位設計

目前欄位如下：

| 欄位     | 用途                             |
| ------ | ------------------------------ |
| 資源ID   | 資源唯一編號，例如 RS000001             |
| 分類     | 顧問資源分類                         |
| 標題     | 顯示於顧問業務大廳的資源名稱                 |
| 簡介     | 資源簡短說明                         |
| 檔案連結   | Google Drive、YouTube、網站或其他資源連結 |
| 資源類型   | 文件、圖片、影片、連結、文字                 |
| 是否顯示   | 是否在顧問業務大廳前台顯示                  |
| 權限類型   | 顧問限定或公開連結                      |
| 排序     | 顯示排序，數字越小越前面                   |
| 最後更新時間 | 資源最後更新日期                       |

僅開放已簽約、已授權的顧問查看。

適用資源包含：

```text

常用連結：合約與收款
獎金結算表
官方 LINE 教學影片
最新消息／公告事項
```

#### 2. 公開連結

可由顧問轉發給會員、潛在會員或企業窗口。

適用資源包含：

```text

健康專欄文章
```

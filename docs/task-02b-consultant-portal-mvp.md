# Task 02B：顧問業務大廳 MVP 完成紀錄、暫緩項目及後續規劃

## 一、任務背景

「元馨醫管家」目前透過健康顧問招募會員，並希望逐步建立一套高效、自動化、可擴充的顧問業務管理系統。

Task 02A 已完成推薦碼追蹤與 LINE 綁定 Webhook。
Task 02B 的主要目標，是在既有 LIFF 與 Apps Script 架構上，建立「顧問業務大廳 MVP」，讓已開通顧問權限者可以透過 LIFF 查看自己的推薦名單與相關業務資料。

本階段重點是先完成「顧問查詢系統的 MVP 基礎」，不是立即取代現有的雲端共享資料夾。

---

## 二、Task 02B 任務目標

Task 02B 的 MVP 目標如下：

1. 顧問可透過專屬 LIFF 入口進入顧問業務大廳。
2. 系統可依 LINE userId 辨識顧問身份。
3. 已開通顧問權限者可看到自己的推薦名單。
4. 系統可支援後續擴充：

   * 我的推薦顧問
   * 導師查看旗下顧問推薦會員
   * 團隊發展概況
   * 顧問資源中心
   * 獎金與組織儀表板

## 五、暫緩項目

### 1. LINE 官方帳號「啟動業務大廳」關鍵字入口暫不修改

目前官方 LINE 的「啟動業務大廳」仍連到原本的雲端共享資料夾。

暫緩原因：

* 現有共享資料夾內仍有顧問需要使用的資料。
* 內容包含招募資料、獎金制度、培訓素材、健康專欄、行銷工具與其他業務文件。
* 若現在直接改成 LIFF 顧問業務大廳，顧問會看不到原本共享資料夾中的其他資料。
* 因此目前先保留既有流程。

後續處理原則：

```text
等顧問業務大廳完整版完成後，再一次性替換 LINE 官方帳號「啟動業務大廳」入口。
```

未來理想狀態：

```text
啟動業務大廳

進入顧問專屬工作台
↓
同時可查看：
1. 我的推薦名單
```

### 健康報告系統後續決策

目前只完成一般會員 Demo 首屏品牌升級。

現有健康報告、身分驗證及報告上傳頁暫不繼續進行局部視覺修改，原因如下：

* 現有上傳流程缺少清楚的逐步操作引導
* 返回 LINE 上傳檔案後，缺少送出成功確認及案件狀態
* 報告收件、AI 分析、製作及 LINE 回傳仍高度依賴人工操作
* 尚未完成從會員上傳到收到分析報告的完整流程驗證
* 現階段局部改色可能在未來重建報告系統時造成重工

後續將另立「健康報告智慧自動化系統」任務，重新規劃：

* 分步操作引導
* 上傳成功確認
* 報告案件與處理進度
* 檔案接收及管理
* AI 分析流程
* 報告產製
* LINE 自動回傳
* 例外狀況及人工接手機制

在報告系統重建前，不再修改現有健康報告、身分驗證及上傳流程。
